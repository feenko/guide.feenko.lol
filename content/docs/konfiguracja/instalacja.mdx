## Wybór silnika

Wybór silnika to kwestia indywidualna, ponieważ każdy ma swoje preferencje. Dlatego warto najpierw zapoznać się z różnymi opcjami, aby lepiej zrozumieć, który silnik będzie dla Ciebie najlepszy.

### Popularne opcje

- Paper
- Pufferfish
- Purpur
- Spigot

Te silniki wspierają najnowsze wersje Minecraft, dzięki czemu masz pewność, że rozgrywka będzie stabilna.

## Instalacja

1. Upewnij się, że masz zainstalowane JDK zgodne z wersją Minecrafta.

2. Utwórz folder o nazwie dla swojego serwera, na przykład `fajnemc`. Nazwa folderu może być dowolna i nie musi odpowiadać nazwie serwera.

3. Pobierz plik JAR serwera z oficjalnej strony silnika, który wybrałeś. Przenieś ten plik do folderu serwera i zmień jego nazwę na `server.jar`.

4. Uruchom serwer tą oto komendą:

```bash
java -Xmx1024M -Xms1024M -jar server.jar
```
